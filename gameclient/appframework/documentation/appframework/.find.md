#.find(selector)

```

Searches through the collection and reduces them to elements that match the selector
      
```

##Example

```
      $("#foo").find(".bar");
      $("#foo").find($(".bar"));
      $("#foo").find($(".bar").get(0));
      
```


##Parameters

```
selector                      String|Object|Array

```

##Returns

```
Object                        an appframework object filtered
```
