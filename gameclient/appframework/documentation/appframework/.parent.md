#.parent(selector)

```

Returns the parent nodes of the elements based off the selector
      
```

##Example

```
      $("#foo").parent(".bar");
      $("#foo").parent($(".bar"));
      $("#foo").parent($(".bar").get(0));
      
```


##Parameters

```
[selector]                    String|Array|Object

```

##Returns

```
Object                        appframework object with unique parents
```
