#.text([text])

```

Gets or sets the innerText for the collection.
If used as a get, the first elements innerText is returned
      
```

##Example

```
      $("#foo").text(); //gets the first elements text;
      $("#foo").text("new text"); //sets the text
      
```


##Parameters

```
text                          String

```

##Returns

```
Object                        an appframework object
```
